# 🇮🇪 Dublin Unemployment Forecaster

An end-to-end data science application analysing and forecasting 
Dublin unemployment trends using 25 years of real Irish government 
data (CSO Ireland, 2000–2025).

## 🎯 Project Overview
This project answers a real-world question:
**"What should Dublin employers expect from the labour market 
over the next 12 months?"**

## ✨ Features
- 📈 25 years of Dublin vs National unemployment trend analysis
- 🤖 Three ML models trained and compared:
  - ARIMA — Best RMSE: 0.77 ⭐
  - Prophet — Seasonal forecasting
  - Neural Network (LSTM) — Deep learning
- 📊 Interactive Plotly visualisations
- 💼 Dynamic business report for Dublin employers
- ⚙️ Configurable forecast horizon (1–8 quarters)

## 🛠️ Tech Stack
Python | Pandas | Statsmodels | Prophet | TensorFlow | Plotly | Streamlit

## 🚀 How to Run
pip install -r requirements.txt
streamlit run app.py

## 📂 Data Source
Central Statistics Office Ireland / data.gov.ie
Quarterly data: Q1 2000 – Q4 2025
