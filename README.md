# Dublin Unemployment Forecaster
An end-to-end forecasting platform analysing Dublin unemployment trends 
using real Irish CSO data (2000-2025).

## Features
- 25 years of Dublin vs National unemployment analysis
- Three ML models: ARIMA, Prophet, Neural Network (LSTM)
- Interactive Streamlit web application
- Dynamic business report for Dublin employers

## How to Run
pip install -r requirements.txt
streamlit run app.py

## Data Source
Central Statistics Office Ireland / data.gov.ie
